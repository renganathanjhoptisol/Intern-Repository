# System Requirements 

	1. Windows OS 10/8/7/Vista/2003 (32 or 64 bit)
	2. Mac OS 10.8.5 or higher
	3. GNOME or KDE desktop
	4. Java JDK5 or later version
	5. Java Runtime Environment
	6. Android Studio
